# Profile Banner Creator

Aqui lhe apresento um dos meus projetos que será incluído no meu portifólio

Atualmente ele contém as seguintes funcionalidades:

v0.1
-- Editar fonte

-- Editar texto
-- Editar cor de fundo
-- Editar tamanho da fonte

v0.2 - adições futuras
-- Escolher tamanho e tipo de banner (youtube, instagram, linkedin etc...)

-- Edições mais avançadas pro texto e mais opções de fontes
## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
